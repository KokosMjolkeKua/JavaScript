INPUT ATTRIBUTES:
value: provides and initial value for the input field
• readonly: input field is read only, but will be sent with the form
• disabled: input field is disabled and not sent with the form
• size: specifies the visible with in characters
• maxlength: specifies the max number of characters allowed
• min and max: specifies the min and max values allowed
• multiple: allows to enter more than one value
• pattern: provides a pattern to check the input value against
• placeholder: provides a short hint displayed in the input field
• required: specifies that input field cannot be empty
• step: provides allowed intervals for an input field
• autofocus: specifies that field must be focused on page load
• height and width: specifies height and width
• list: refers to a datalist element with pre-defined options
• autocomplete: turns on and off the autocomplete for a field
